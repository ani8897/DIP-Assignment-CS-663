Linear Contrast Stretching

Formula applied for Linear Contrast Stretching:

 I_new(x,y) = (I_old(x,y) - min(I_old)) / (max(I_old) - min(I_old))
Observations for Image(5): 
There isn't much change. Linear Contrast stretching is not effective here as for every channel, 
the minimum (0) and maximum (255) pixel intensities are already present and hence linear mapping does not result in any change according to the formula

Histogram Equalization

Observations for image(5):
There is a noticeable change in the contrast of the image. 
We can clearly see the parts in new image which were underlit in the original image and hence we would prefer Histogram Equalization for this image
